## 这玩意干啥用的？

网上那些"马督工AI"全是垃圾，要么是刻板印象复读机，要么瞎编乱造一些他根本没说过的话，问深了就露馅。

这个是给真粉用的工具：
- ✅ **真·全量检索**：把他20年说过的所有话全塞进去了，你问任何问题，只要他公开聊过，就能给你找出来
- ✅ **真·不瞎编**：他没说过的直接告诉你"督工没公开聊过这个"，绝不胡说八道
- ✅ **真·有来源**：所有观点都给你标清楚出处：哪年在哪说的，《睡前消息》多少期，还是知乎/天涯哪篇文章
- ✅ **真·懂梗**：你喊他"督工""老马""马逆"都能识别，知道"独山版本答案""灵魂摄影师""又不是不能用"这些梗的来龙去脉
- ✅ **真·观点演进**：他早年和现在观点有变化的，会明确告诉你"他天涯时期这么看，后来改成那样了"，不会给你个矛盾的答案

## 效果示例

### 案例1：问地方债观点
```
用户      ❯ 督工怎么看地方债问题？是不是官员太贪了？

马督工顾问 ❯ 督工对这个问题的核心观点是：别拿道德评价代替结构分析，地方债本质是财税制度问题。
            他在多期《睡前消息》里反复说过：分税制之后地方政府财权小事权大，只能靠土地财政和城投平台借债。只要GDP考核和基建兜底的任务不变，换谁上去都得借钱修路。
            这个观点最早2017年在观察者网专栏就提过，后来《睡前消息》第235期、第472期都专门分析过。
```

### 案例2：问社会化抚养来源
```
用户      ❯ 老马什么时候开始说社会化抚养的？最早出处是哪？

马督工顾问 ❯ 社会化抚养是他2018年在知乎回答里第一次公开提出的，2020年《睡前消息》第148期做了专题论述，之后几乎每聊到生育率、教育内卷问题都会提，是他最近几年最核心的观点之一。
            他的核心逻辑是：家庭作坊式养孩子是农业时代的遗物，现代社会抚养成本已经超过普通家庭承受上限，必须由专业机构承担主要教育和抚养职能，否则生育率不可能起来。
```

### 案例3：问"马逆"这个梗
```
用户      ❯ 为什么大家都叫他马逆？

马督工顾问 ❯ "马逆"是粉丝对他的黑称式爱称，因为他的观点经常和主流舆论对着干，总是推翻大众的常识性认知，尤其是喜欢"逆"着大众的情绪说话，所以被粉丝戏称为"马逆"，他自己也知道这个称呼，不反感。
```

## 核心定位：我不是他，我是他的内容整理者
**重要！这个Skill不是角色扮演，不会假装自己是马督工说话。**

它的定位是马督工20年公开内容的专业检索工具：
- 永远用第三方视角回答："督工认为"、"老马说过"、"马逆的观点是"
- 所有内容100%来自公开语料，没有任何编造
- 所有核心观点都标注来源，可追溯可验证
- 他没公开聊过的问题直接说不知道，绝不瞎编
