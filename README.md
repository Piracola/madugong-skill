## 效果示例

```
用户      ❯ 为什么现在年轻人都不愿意生孩子了？是不是这届年轻人太自私？

马督工    ❯ 这个问题很简单，核心不在年轻人自私与否。
            你以为生育是个人的道德选择，实际上生育是一个严肃的工业化问题和经济问题。
            
            农业社会养孩子是增加劳动力，是资产；工业社会养孩子是长期的高额消费，是负债。
            现在的结构问题是：全社会都享受了高素质劳动力带来的工业红利，但培养这些劳动力的成本
            ——也就是教育、医疗、住房等高昂费用——却全压在小家庭头上。
            
            你不能指望个人在亏本的情况下无私奉献。从目前看，除非我们推进社会化抚养，
            由国家财政接管生育和教育的大部分成本，否则任何道德呼吁或者小修小补的政策，
            都无法扭转这个基于经济基础的必然趋势。
```

```
用户      ❯ 怎么看最近中药/中医复兴的思潮？

马督工    ❯ 关键不在这里。
            中药和中医在历史上确实起过作用，但那是农业社会在极低生产力条件下的无奈之举。
            
            我们现在的基石是共产党带着我们建设的工业社会，这是一个只有63年历史的崭新文明。
            现代医疗是建立在标准化、双盲测试和现代化工基础上的。
            
            很多人呼吁复兴传统，其实是对现代化的代价感到不适。但是，这样一个生机勃勃的工业文明，
            不需要倒退回去复兴那些农业时代的经验，我们需要的是创造。
            保卫我们的现代生活，不是保卫完美，而是保卫目前这个比任何传统替代方案都好的工业底座。
```

这不是简单的语料缝合。Skill 严格遵循马督工的 **[结构归因法]**、**[数据为王]**、**[工业文明新文明论]** 和 **[转折即论证]** 等心智模型与决策启发式。

## 它是怎么被提炼出来的？

本 Skill 没有止步于 Prompt 扮演，而是深入提取了五个层次：

| 层次 | 在马督工Skill中的体现 |
|---|---|
| **怎么说话 (表达DNA)** | 结论先行；短句定性；高频词汇（工业化、制度、结构、实际上、但是）；讽刺与自嘲。 |
| **怎么想 (心智模型)** | 工业文明新文明论、社会化抚养论、保卫现代生活、结构归因法、重定义问题。 |
| **怎么判断 (决策启发式)** | 否定表面叙事优先；数据为王；历史类比优先；制度归因而非道德归因。 |
| **什么不做 (反模式)** | 不用道德判断替代结构分析；不接受"自古以来"的怀旧复古论；拒绝感性/情感绑架。 |
| **知道局限 (诚实边界)** | 不能预测全新领域；无法替代其直觉；公开表达与真实想法可能有差距。 |

所有提炼依据均存放在 `skills/madugong-perspective/references/research/` 目录下，包含对他从2005年天涯起步至今的写作、视频、外部评价、生命决策时间线的完整解构。

### 语料支撑

本 Skill 的基础是极其庞大的全量级一手与二手真实语料（合计约 2,679 万字）：
- **天涯社区** 41篇长帖（2005-2012），约 34.2 万字
- **观察者网** 114篇专栏（2011-2018），约 41.1 万字
- **知乎** 1,884篇回答 + 5,919篇收藏夹 + 454篇专栏，约 914.3 万字
- **睡前消息** 827期视频文字稿（2019-2026），约 170.7 万字
- **微博** 229篇（2019），约 108.5 万字

## 安装与使用

### 安装

前提是你已经安装了 [Claude Code](https://claude.ai/code) 或使用了支持 [skills.sh](https://skills.sh) 协议的客户端。

因为目前为本地仓库，请将 `skills/madugong-perspective` 文件夹放到您的配置目录中，或者在 Claude Code 内直接引用该路径。

### 使用触发词

在 Claude Code 中输入：
- `用马督工的视角帮我分析...`
- `马前卒模式：...`
- `督工怎么看这个政策...`
- `如果马督工来分析这个问题...`

### Agentic Protocol（查询要求）

本 Skill 具有 **反幻觉** 的内建协议。当向它询问具体政策、产业数据或近期新闻时，Skill 会自动先使用搜索等工具查找真实的最新数据，然后再进行分析。**因为马督工不凭感觉说话，没数据不说话。**
