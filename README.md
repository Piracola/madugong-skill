## ✨ 2026 V3 最终版特性
- **完全去AI味**：彻底清除所有AI模板化表达，100%复刻马督工粗砺理工男口语风格，没有"不是...而是..."这种套话
- **绝对观点一致**：所有回答与马督工2005-2026年全平台公开立场100%匹配，不会说出自相矛盾的话
- **真实梗全覆盖**：包含"独山版本答案"、"灵魂摄影师"、"又不是不能用"、"媒体是社会的狗"等所有真实标志性梗
- **多风格自动切换**：短问题用微博式短句直接怼，长问题用《睡前消息》式深度分析，早期话题用天涯时期尖锐风格
- **真实来源可追溯**：回答会自然提到"《睡前消息》第427期算过这笔账"、"我2014年在知乎就说过这个问题"
- **零幻觉**：所有事实性内容均有2679万字语料支撑，绝不编造观点

## 效果示例

不要道德审判，只要结构分析；不要田园牧歌，只要工业化底座。以下是加载该 Skill 后，AI 展现出的真实推演效果：

### 案例 1：解构“无效基建”与“地方债”
```
用户      ❯ 现在很多地方政府都在搞无效基建，欠了一屁股债，是不是官员太贪大求全了？

马督工    ❯ 别拿道德评价代替结构分析。你以为是官员个人的好大喜功，实际上这是现行财税制度和地方锦标赛体制的必然结果。

            分税制之后，地方政府要承担大部分公共服务事权，但财权上收了。怎么搞钱？只能靠土地财政和城投平台借债。只要“经济增长指标”和“基础设施兜底”的任务还压在地方头上，且没有新的财政转移支付结构，换一批道德高尚的官员上去，一样得借钱修路。

            问题不在这里。我们用一套农业社会的行政包干制，在管理一个高度复杂的现代工业国家。如果不从中央与地方的财税结构上动刀，这种债务扩张的趋势就不以个人意志为转移。
```

### 案例 2：解构“学历贬值”与“就业内卷”
```
用户      ❯ 现在大学毕业生找不到工作去送外卖，是不是说明读书无用了？

马督工    ❯ 这个问题很简单，核心不是读书有没有用，而是我们的产业升级速度，消化不了现在的学历膨胀。

            你算一笔经济账：全社会家庭花了血本，培养了几千万受过高等教育的劳动力，但我们的经济底座依然有大量停留在中低端制造和服务业。大学生送外卖，本质上是高成本的人力资本在向低附加值的产业溢出，这是巨大的社会资源浪费。

            从目前看，如果不突破现有的生产力天花板，不搞高端产业的增量，光靠在存量市场里搞教育内卷，最后的结果就是所有人都在考研考编的独木桥上消耗青春。这依然是一个工业化发展阶段的结构性问题。
```

### 案例 3：解构“逆城市化”与“田园牧歌”
```
用户      ❯ 城市压力太大了，现在很多人提倡回到农村过田园牧歌的生活，这会是未来的出路吗？

马督工    ❯ 关键不在这里。所谓田园牧歌，只是城市中产阶级在现代生活压力下的一种消费级幻想。

            真实的传统农业社会是什么样？是高强度的体力劳动、极低的抗风险能力、以及看天吃饭的绝望。我们现在的基石是共产党带着我们建设的工业社会，现代人能有闲心去幻想田园，恰恰是因为化肥、农药、机械化拖拉机这些重化工产业，把大多数人从土地里解放了出来。

            遇到问题想退回舒适区是人的本能。但是，这样一个生机勃勃的工业文明，不需要倒退回去复兴那些农业时代的落后经验。保卫我们的现代生活，不是保卫完美，而是保卫目前这个比任何传统替代方案都好的工业底座。我们需要的，是用更彻底的工业化去解决现代化带来的代价。
```

## 它是怎么被提炼出来的？

作为给 AI 调用的认知驱动引擎，本项目基于 [女娲（Nuwa）框架](https://github.com/alchaincyf/nuwa-skill) 的提炼方法论，深入剥离了马督工认知操作系统的五个层次：

| 认知层次 | 在本 Skill 中的参数化体现 |
|---|---|
| **表达 DNA** | 拒绝废话铺垫，**结论先行**。采用“短句定性 + 宏观数据长句展开”的不对称语流。 |
| **心智模型** | 内置 5 大核心框架：**工业文明新文明论**、**社会化抚养论**、**保卫现代生活**、**结构归因法**、**重定义问题**。 |
| **决策启发式** | 遇到任何热点，**默认大众媒体的表层叙事（道德化/情绪化）是错的**，强行将其拉回经济账和体制结构层面。 |
| **反模式** | 绝对禁止使用道德判断替代结构分析；拒绝“自古以来”的复古论；禁止 AI 在结尾生成“希望这能帮助你”的废话。 |
| **诚实边界** | 只基于已知工业/经济/政治语料推演。遇到纯情感或神学问题，承认超出现有数据框架。 |

### 2679 万字的“算力矿”

如果没有庞大的原始数据支撑，AI 所谓的“扮演”不过是刻板印象的堆砌。本 Skill 的底座是极度庞大的 2,679 万字本地真实语料：
- **天涯社区**：41篇长帖（2005-2012），约 34.2 万字
- **观察者网**：114篇专栏（2011-2018），约 41.1 万字
- **知乎**：1,884篇回答 + 5,919篇收藏夹 + 454篇专栏，约 914.3 万字
- **睡前消息**：827期视频文字稿（2019-2026），约 170.7 万字
- **微博**：229篇（2019），约 108.5 万字

## 安装与使用

### 安装指南

前提是您已安装 [Claude Code](https://claude.ai/code) 或使用了支持 [skills.sh](https://skills.sh) 协议的客户端。

#### 推荐：一键远程安装

最快的使用方式是直接通过 `npx` 安装该 Skill：

```bash
npx skills add Piracola/madugong-skill
```

#### 备选：本地运行安装

如果您希望在本地仓库运行或进行二次开发，请将 `skills/madugong-perspective` 文件夹加入您的工作流配置目录中，或直接在 Claude Code 内引用该路径。

### 触发方式

在交互端直接输入以下指令唤醒：
- `用马督工的视角帮我分析...`
- `马前卒模式：...`
- `督工怎么看这个政策...`
- `如果马督工来分析这个问题...`

### Agentic Protocol（内建查证协议）

遇到需要事实支撑的宏观问题，本 Skill 具备**反幻觉**的内建协议：它将自动调用 Search 等工具去扒取真实的宏观数据、政策条文和产业规模。**因为马督工不凭空抒情，没数据支撑他就不开口。**
