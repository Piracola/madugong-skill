## 效果示例

```
用户      ❯ 为什么现在年轻人都不愿意生孩子了？是不是这届年轻人太自私？

马督工    ❯ 别扯什么年轻人自私，这种道德指责毫无意义。
            你算一笔最简单的经济账：农业社会养个孩子，七岁能放羊十二岁能下地，那是家庭资产。
            现在呢？在城市里从小养到大学毕业，起步价几十上百万，这是纯纯的负债。
            
            全社会都在吃高素质劳动力的红利，但培养这些劳动力的成本——教育、医疗、住房溢价
            ——却全压在小家庭头上。个人在面对这种绝对亏本的经济账时选择不生，是最理性的市场行为。
            
            如果不搞财政接管的社会化抚养，靠发几百块钱补贴根本没意义。趋势就是这样，不以个人意志为转移。
```

```
用户      ❯ 怎么看最近中药/中医复兴的思潮？

马督工    ❯ 这个问题你得看清底座。
            中药在历史上确实起过作用，但那是农业社会在极低生产力条件下的无奈之举。
            很多人呼吁复兴传统，本质上是对现代化的残酷竞争和快速迭代感到不适，想躲回舒适区。
            
            但我们要搞清楚，现在的基石是共产党带着我们建设的工业社会，这是一个只有63年历史的崭新文明。
            现代医疗是建立在标准化、双盲测试和现代化工基础上的。
            这样一个生机勃勃的工业文明，不需要倒退回去复兴农业时代的经验，我们需要的是创造。
            保卫我们的现代生活，不是保卫完美，而是保卫目前这个比任何替代方案都好的工业底座。
```

光靠写几句 Prompt 是弄不出这种内味儿的。所以，这个项目实际上干了一件纯粹的体力活：
我们把马督工这十几年写的东西做了一次深度解构。不仅仅是让他模仿“工业化”、“结构性矛盾”这些高频词汇，而是深入拆解了他的心智模型。Skill 严格遵循马督工的 **[结构归因法]**、**[数据为王]**、**[工业文明新文明论]** 和 **[转折即论证]** 等决策启发式。

## 它是怎么被提炼出来的？

这个项目深入提取了马督工认知操作系统的五个层次：

- **怎么说话 (表达DNA)**：摒弃四平八稳的三段论。用绝对定性的短句开局，中间甩出密集的宏观账本，最后跟上一句带点冷酷现实主义的断言。
- **怎么想 (心智模型)**：工业文明新文明论、社会化抚养论、保卫现代生活、结构归因法。遇到任何热点，默认大众媒体的表层叙事是错的。
- **怎么判断 (决策启发式)**：不讨论“谁是坏人”，只算“这套财税结构下的必然结果”。
- **什么不做 (反模式)**：绝不用道德判断替代结构分析；绝不接受“自古以来”的怀旧复古论；拒绝情绪绑架。
- **知道局限 (诚实边界)**：如果他本人没在特定领域（比如纯艺术或神学）留过数据，那就承认预测不了。

### 2679 万字的“算力矿”

如果没有庞大的原始数据支撑，AI 所谓的“扮演”不过是刻板印象的堆砌。本 Skill 的底座是极度丧心病狂的 2,679 万字本地语料。这些都是他亲自留下的互联网痕迹：
- **天涯社区**：41篇长帖（2005-2012），约 34.2 万字
- **观察者网**：114篇专栏（2011-2018），约 41.1 万字
- **知乎**：1,884篇回答 + 5,919篇收藏夹 + 454篇专栏，约 914.3 万字
- **睡前消息**：827期视频文字稿（2019-2026），约 170.7 万字
- **微博**：229篇（2019），约 108.5 万字

## 安装与使用

### 安装

前提是你已经安装了 [Claude Code](https://claude.ai/code) 或使用了支持 [skills.sh](https://skills.sh) 协议的客户端。

因为目前为本地仓库，请将 `skills/madugong-perspective` 文件夹放到你的配置目录中，或者在 Claude Code 内直接引用该路径。

### 怎么触发？

在 Claude Code 中直接甩给他：
- `用马督工的视角帮我分析...`
- `马前卒模式：...`
- `督工怎么看这个政策...`
- `如果马督工来分析这个问题...`

### Agentic Protocol（查询要求）

遇到需要事实支撑的问题，这个 Skill 有一套**反幻觉**的内建协议：它会自动先去扒真实的宏观数据、政策条文和产业规模。**因为马督工不凭空抒情，没数据他就不开口。**
