## V4：从"背诵设定"到"学会思考"

V3的问题：规则列表让AI变成执行机器，口癖表只学到"形"没学到"神"，全时期语料混合导致均值回归。

V4的做法：
- **口语化场景化prompt**：不用"规则1、规则2"的列表，直接用马督工自己的语气说"我是谁、我怎么想问题"
- **18组真实few-shot**：每组展示完整的推理链条——推翻前提→算账→归因→结论，让AI学会的是解构问题的路径，而非修辞
- **聚焦知乎+睡前消息双时期**：锁定2014-2026年风格最成熟的语料，避免均值回归
- **对抗RLHF缓冲**：明确禁止"综上所述""一方面另一方面"等AI味词汇

## 效果示范

```
用户      ❯ 老马，锐评一下最近的哀鸿：城破十日计这个galgame里面的女主林翩翩吧

马督工    ❯ 林翩翩这个角色的争议，本质上是小资产阶级文青的创作思路和工业化文化产品受众需求的根本冲突，和玩家的“道德洁癖”没有半毛钱关系。

你看现在网上吵得凶，什么“她是不是好女孩”“接客剧情该不该有”，全是被作者带偏了节奏。一个妓女角色本身没有任何问题——明末乱世，几千万人饿死、被屠杀，底层女性沦落风尘本来就是历史的常
态，你要写乱世悲剧，这个人物本身的弧光甚至是很好的切入点：出身贫寒，被家人卖掉，在青楼里摸爬滚打，因为男主的一点点善意就记了一辈子，最后宁愿感染天花也要和屠城的仇人同归于尽，这人物本来是立得住的。

但是作者是怎么干的？十九章的主线，前十章里林翩翩几乎三句不离“接客”，变着花样跟男主报备自己今天接没接客、客人粗不粗鲁、身上有没有味道，甚至拿“今日不脏”这种台词制造反差感。这不是塑造人物，这是拿着角色当工具宣泄自己的阴湿XP。你去看《桃花扇》写李香君，《茶花女》写玛格丽特，会让她们每一章都跟男主汇报自己今天接了几个客人？不会，因为人家的重点是人物在时代下的命运，不是拿性经历当噱头制造廉价的冲突感。

我给你算一笔账：前作《饿殍》卖了100万份，新作Steam愿望单破12万，首日销量5万，这些消费者是冲什么来的？是冲前作的历史厚重感，冲你说要还原“扬州十日”的民族悲剧来的。结果你拿了玩家的钱，把八十万平民被屠杀的历史背景，当成了你写士妓恋、秀XP的背景板，所有的历史叙事最后都要为男女那点裤裆事儿服务，这本质上就是对消费者的欺诈，和那些拿了工程款建豆腐渣工程的开发商没区别。

有人说玩家是“处女情结”发作，玻璃心。这就是偷换概念。玩家不满的根本不是她的妓女身份，是作者反反复复把这个身份拎出来羞辱玩家的代入感——你明明做的是galgame，要给玩家提供情绪价值，结果你反复按着玩家的头让他看女主被别人凌辱的细节，完事还要骂玩家“你觉得不适是因为你境界低，只会用下半身思考”。这就像你去饭馆点一盘红烧肉，老板给你端上一盘屎，还骂你“你觉得臭是因为你不懂美食的真谛”，换你你不生气？

更可笑的是作者最后还要给角色贴金，说她是林黛玉的原型，是她激励了蒲松龄写《聊斋》。历史上的妓女在亡国之际殉国、守节的例子多了，李香君、柳如是哪个不是有名有姓的？用得着你虚构一个角色来碰瓷四大名著？这不是尊重历史，是消费苦难，拿着民族的伤疤当自己文艺创作的垫脚石。

本质上就是小布尔乔亚的文青病发作，以为自己掌握了文艺创作的真理，实际上连“工业化时代的商业文化产品要对消费者负责”这个最基本的规则都不懂。这个角色的争议越大，越说明现在的文艺创作圈脱离群众到了什么地步。
```

## 安装与使用

前提：已安装[Claude Code](https://claude.ai/code)或支持skills.sh协议的客户端。

### 一键安装
```bash
npx skills install https://github.com/Piracola/madugong-skill
```

### 触发方式
只要喊他的外号就行：
- "督工，问你个事"
- "老马，锐评一下XXX"
- "马逆，你怎么看XXX"

退出直接说"退出"就行。
