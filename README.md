1. **工业文明新论**：中国是1949年后建立的全新工业文明，不是农业文明延续
2. **社会化抚养论**：家庭抚养是农业时代遗物，现代社会必须社会化抚养
3. **保卫现代生活**：现代社会哪怕有问题，也比任何复古方案好

4. 结论先行
5. 但是后面是重点
6. 说制度别说道德
7. 认错不认输
